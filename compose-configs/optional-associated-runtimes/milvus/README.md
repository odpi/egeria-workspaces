# Overview
This directory contains a sample Docker Compose script to support the deployment of Milvus for experimentation,
development, and learning. [Milvus](https://github.com/milvus-io/milvus) is an open source vector database designed to 
support AI applications. We are using Milvus in our AI experiments in the Egeria project.


As always, your feedback and participation are welcome. 

# Milvus Admin UI
Milvus provides a web-based user interface for managing and interacting with the database. You can access the Milvus Admin UI at http://localhost:9091/webui

# Milvus access
Milvus is at http://localhost:19530
