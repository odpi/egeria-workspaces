This file creates the **Common** top-level folder in the Data Field Naming glossary. It groups the
Common Modifiers and Common Class Words folders, which hold modifier and class word terms that are
generic enough to be reused across many subject areas, rather than being repeated in every subject
area's own Modifiers or Class Words folder.

Requires `glossary.md` to have been processed first.

____

## Create Collection Folder

### Display Name

Common

### Qualified Name

CollectionFolder::DataFieldNaming:Common

### Purpose

Groups the folders holding modifiers and class words that are generic enough to be reused across
multiple subject areas.

### Parent ID

Glossary::DataFieldNaming

### Parent Relationship Type Name

CollectionMembership
