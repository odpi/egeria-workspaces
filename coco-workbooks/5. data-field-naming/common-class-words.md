This file creates the **Common Class Words** folder in the Data Field Naming glossary, along with class word
terms that are generic enough to be reused across many subject areas, so they don't need to be repeated in
every subject area's own Class Words folder.

Requires `glossary.md` and `common.md` to have been processed first.

____

## Create Collection Folder

### Display Name

Common Class Words

### Qualified Name

CollectionFolder::DataFieldNaming:CommonClassWords

### Purpose

Class words that are generic enough to be reused across multiple subject areas when building data field names.

### Parent ID

CollectionFolder::DataFieldNaming:Common

### Parent Relationship Type Name

CollectionMembership
