This file creates the **Common Modifiers** folder in the Data Field Naming glossary, along with modifier
terms that are generic enough to be reused across many subject areas, so they don't need to be repeated in
every subject area's own Modifiers folder.

Requires `glossary.md` and `common.md` to have been processed first.

____

## Create Collection Folder

### Display Name

Common Modifiers

### Qualified Name

CollectionFolder::DataFieldNaming:CommonModifiers

### Purpose

Modifiers that are generic enough to be reused across multiple subject areas when building data field names.

### Parent ID

CollectionFolder::DataFieldNaming:Common

### Parent Relationship Type Name

CollectionMembership
