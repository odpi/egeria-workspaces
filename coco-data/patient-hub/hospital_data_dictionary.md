## medical_image

### Display Name
medical_image

### In Data Dictionary
DataDictionary::PatientRecordDataHub

### Description
Metadata and file references for medical imaging studies including X-rays, CT, MRI, ultrasound, and nuclear medicine. Image files are stored on a PACS server, network storage, or object store; this table holds the metadata and pointer to each file. Includes DICOM standard identifiers for interoperability with PACS and radiology information systems.

### Qualified Name
DataStructure::medical_image

### Version Identifier
1.0

## audit_log

### Display Name
audit_log

### In Data Dictionary
DataDictionary::PatientRecordDataHub

### Description
A complete, immutable record of every INSERT, UPDATE, and DELETE across all clinical tables. Rows are written automatically by the fn_audit_trigger() database trigger; application code does not write to this table directly. Stores the full before and after state of each changed row as JSONB.

### Qualified Name
DataStructure::audit_log

### Version Identifier
1.0
